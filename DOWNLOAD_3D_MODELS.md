## **TROUBLESHOOTING**

**Q: Model not showing?**
A: Check console (F12) for errors. Verify file path.

**Q: Model too big/small?**
A: Edit `public/js/jewelry-3d-renderer.js` and adjust scale.

**Q: Model in wrong position?**
A: Edit `public/js/jewelry-3d-manager.js` and adjust offsets.
